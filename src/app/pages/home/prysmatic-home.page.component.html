Prysm Specifications

Network

Ethereum Serenity, Shasper

(Sharding + Casper Proof of Stake)
Implementation Language Golang
networking protocol Libp2p from Protocol Labs
Interprocess Communication Google's gRPC framework
Test and Build Pipeline Google's Bazel build tool
Database BoltDB, a write-optimized, persistent key-value store
